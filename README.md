# RCB IPL Performance Analysis 

# 📌 Project Overview

This project analyzes Royal Challengers Bangalore (RCB) across 10+ IPL seasons using advanced SQL and Excel-based insights.
The goal is to understand RCB’s team performance, player trends, venue impact, and strategic gaps to support better auction strategy, team selection, and match-day planning.

# 🎯 Objective

Evaluate RCB’s performance consistency across multiple seasons
Identify strengths/weaknesses in batting, bowling, and middle-order stability
Derive KPIs to measure players, venues, and game situations
Convert raw IPL data into actionable insights for strategic decision-making

# 🛠️ Tools & Technologies

SQL – joins, aggregations, window functions, progression metrics
Excel – charts, pivot analysis, trend visualization
Datasets Used: Matches, Ball-by-Ball, Players, Seasons, Venues, Wickets, Player Roles (20+ tables)

# 📂 Data Preparation & Modeling

Connected 20+ IPL tables using SQL joins
Cleaned inconsistent fields (player names, venue strings, match types)
Created derived metrics like:
Total Runs, Wickets, Strike Rate, Economy
Venue-wise batting & bowling performance
Toss and decision impact
Over-phase metrics (Powerplay, Middle Overs, Death Overs)

# 📊 Key Analyses Performed

1️⃣ Team Performance Analysis
Season-wise runs, wickets, win margins
Consistency trends over 10+ seasons
Identification of best & worst performing seasons

2️⃣ Player Performance Analysis
Top performers via progression-based SQL
Batsmen analysis (strike rate, consistency, boundary %)
Bowlers analysis (economy, death-over performance)
High-value all-rounders

3️⃣ Venue & Toss Impact
Performance across 30+ stadiums
Batting-first vs chasing advantage
Toss win → match win correlation

# 🧠 Key Insights

🔹 Batting inconsistency strongly affected season outcomes
High-scoring seasons aligned with strong finishes
Seasons with <2000 total runs had bottom-four results

🔹 Bowling remained RCB’s weakest area
Lack of strike bowlers → high totals conceded
Seasons with <80 wickets showed poor match momentum

🔹 Middle-order was a major gap
Heavy dependency on Kohli–ABD–Gayle
Lack of reliable finishers at positions 5–7

🔹 Venue & Toss decisions influenced success
Strongest performances at M. Chinnaswamy
Teams batting second after winning toss had higher win %

# 💡 Recommendations

Strengthen middle order with stable finishers
Add specialist bowlers for powerplay, middle overs, and death overs
Invest in high-impact all-rounders
Build a balanced squad with youth + experience
